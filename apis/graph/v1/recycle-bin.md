# Recycle Bin

Base URI: ```https://graph.microsoft.com/v1.0/```

## List items
| Method   | URI |
| :------- | :------- |
| GET | /storage/fileStorage/containers/{containerId}/recycleBin/items |

### Query Parameters
This method supports the $select and $top OData query parameters to customize the response.
